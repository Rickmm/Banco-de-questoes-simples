# Banco-de-quest-es-simples
Projeto simples de quiz em terminal desenvolvido em Python. As perguntas são carregadas a partir de um arquivo CSV e apresentadas ao usuário em ordem aleatória, sem repetição. O programa valida a entrada do usuário, informa se a resposta está correta ou incorreta, e pode ser encerrado a qualquer momento digitando "Sair". Tecnologias: Python, Pandas
